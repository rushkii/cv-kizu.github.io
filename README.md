# Pemrograman Web - UTS

Repository ini bertujuan untuk tugas Ujian Tengah Semester mata kuliah Pemrograman Web I berupa website Curriculum Vitae (CV) sederhana yang tetap terlihat menarik.

## Tentang

Website ini menampilkan profil dan CV saya yang seorang Frontend Developer. Dibangun menggunakan HTML, TailwindCSS v4 (via CDN), dan Vanilla JavaScript.

## Tech Stack

- HTML5
- TailwindCSS v4 (CDN)
- Vanilla JavaScript
- Splide.js (slider/gallery)
- Google Fonts (Inter)

## Fitur

- Halaman profil lengkap (tentang, pengalaman, pendidikan, portfolio, keahlian, sertifikat)
- Dark mode / light mode toggle
- Responsive design
- Gallery showcase aplikasi menggunakan Splide.js slider
- Tabel pendidikan dan portfolio
- Link ke sosial media dan kontak

## Cara Menjalankan

Cukup buka file `index.html` langsung di browser, atau kalau mau pakai local server:

```bash
# klo pake Python bisa dengan jalankan di bawah ini.
python3 -m http.server 8000

# kalau Node.js seperti ini.
npx serve .
```

Lalu buka `http://localhost:8000` di browser.
